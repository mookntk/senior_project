# senior_project

## Getting Started

```
# install dependencies
npm install

# serve with host reload at localhost:8080
npm run start
```

## Database

Database Configuration : @nodeserver/configs/db.js

Database File : @nodeserver/configs/project_db.sql

```
  host: "localhost"
  user: "root"
  password: "root"
  database: "senior_project"
  charset: "utf8"
```
